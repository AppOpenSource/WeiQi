生命如此短暂，为什么总要将青春浪费在不断的选择之中呢？罚你，回头阅读心理学家施瓦茨（Barry Schwartz）的TED演讲：选择之困惑――为何多即是少，1百遍啊1百遍。请记住施瓦茨的演讲要点：

更多的选择不代表更多的自由；
更多的选择导致决策的延迟和降低的满意感；
快乐之秘诀，在于降低自己的期望值。
